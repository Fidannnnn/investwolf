# InvestWolf - Demo Account

InvestWolf is a web-based demo trading platform designed for beginners to explore stock trading without real money. It provides a simple interface to **buy, sell, and track stocks**, manage a watchlist, and simulate investing with a demo balance.

---


## Features

### Account & Balance
- **Demo balance**: Users start with $1000 demo USD.
- **Top Up**: Simulate adding funds at the rate of **1 AZN = $1000 USD**.
- **Invest Popup**: Placeholder modal showing future real investing option.

### Stocks & Watchlist
- **All Stocks Section**: Displays only stocks you currently own.
- **Watchlist**: Pre-seeded with several popular stocks. Users can add new ones.
- **Search**: Search stocks by **symbol or name**, showing a **detailed info card** like:
  - Ticker
  - Sector
  - Industry
  - Current Price
  - Dayâ€™s Range
  - 52-Week Range
- **Stock Modal**: Clicking a stock anywhere opens a modal to:
  - Buy
  - Sell
  - Add to Watchlist
  - See quantity owned
  - See detailed stock information

### UI & Navigation
- Responsive layout, centered for mobile or small screens.
- Bottom navigation linking to:
  - AI page
  - Home
  - Demo Account
- Popups and modals for balance top-up, investing, and stock details.

---

## Technologies
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
- **LocalStorage** for persisting:
  - Stock quantities
  - Watchlist
  - Balance

No backend is required; everything runs in the browser.

---

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. Start exploring:
   - Top up your balance
   - Search and manage stocks
   - Buy, sell, and add to watchlist
